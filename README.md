基于全新架构的 龙脉R(AnaDos) 自动战斗/护肝小助手。图像技术+模拟控制，解放双手！

## 功能列表

- **启动游戏**：自动启动游戏并领取每日奖励，收取邮件(可选)，进入主界面。
- **刷素材**：
  - 支持所有素材关卡。
  - 自定义挑战次数，自动使用大小体力药。
  - 锐意制作中
- **活动**：
  - 锐意制作中
- **领取奖励**：
  - 锐意制作中
- **一键养成**:
  - 锐意制作中
