基于全新架构的 龙脉R(AnaDos) 自动战斗/护肝小助手。图像技术+模拟控制，解放双手！

## 功能列表

- **启动游戏**：
  - 自动启动游戏并领取每日奖励，收取邮件(可选)，进入主界面。
- **刷素材**：
  - 重构中
- **活动**：
  - 当期及复刻活动底关自动使用跳过券清体力
- **兑换活动商店**：
  - 锐意制作中
- **每日精粹商店**：
  - 锐意制作中
- **一键养成**:
  - 锐意制作中
